Wodstorm é uma plataforma na qual o personal trainer irá salvar os treinos para cada um de seus alunos. E os alunos irão utilizar o app para acompanhar seus treinos.
O Wodstorm é feito para todos os esportes. Independente de qual esporte seja, personal trainer irá utilizá-lo para passar os treinos.
O app é 100% gratuito.
